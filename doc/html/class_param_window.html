Detailed Description

Modeluje okno z parametrami algorytmu odejmowania tła.

Klasa opisuje okno, które daje możliwość ustawienia parametrów algorytmu modelowania tła. Opcję, które można ustawić: wypełnianie dziur, rozmiar jądra , wartość progowania, filtrowanie gaussa.
